Plottr is a web-based visualization tool that lets you explore **weather forecasts** around the world.  
You can search any place, view **temperature and precipitation charts**, see the **location on a map**, and even generate a simple **linear forecast** for future values.
 
 Features
 
Live location search with suggestions (powered by Open-Meteo geocoding)
Interactive map using [Leaflet](https://leafletjs.com/)
Hourly weather data from [Open-Meteo](https://open-meteo.com/)
 Temperature and Precipitation charts built with [Chart.js](https://www.chartjs.org/)
 Linear regression prediction for next 6 hours
  Local storage for recent searches
  Planned: CSV export of weather data

 
Frontend: HTML5, CSS3, JavaScript (Vanilla)
Libraries: Chart.js, Leaflet.js
API: [Open-Meteo Weather & Geocoding API](https://open-meteo.com/)
Design: Dark theme with responsive layout
